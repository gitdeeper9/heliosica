# HELIOSICA Data Directory

## Structure
- `validation/` - 312-event validation catalogue
- `thresholds/` - SPIN parameter thresholds
- `calibration/` - DBM and Forbush calibration files
- `sample/` - Sample data for testing
- `raw/` - Raw data from external sources
  - `soho/` - SOHO/LASCO CME catalogue
  - `dscovr/` - DSCOVR real-time data
  - `omni/` - OMNI heliospheric dataset
  - `nmdb/` - Neutron monitor data

## File Formats
- HDF5 for catalogue data
- NetCDF4 for time series
- CSV for thresholds and calibration
